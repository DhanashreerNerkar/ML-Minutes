â ï¸ Common Misconceptions About RÂ²

â Myth 1: Higher RÂ² = Better Model

Reality: You can artificially increase RÂ² by adding more variables (even random ones!). This leads to overfitting.

Always use Adjusted RÂ² when comparing models with different numbers of features.

â Myth 2: RÂ² = 0.90 means 90% accurate predictions

Reality: RÂ² measures explained variance, not prediction accuracy. Your predictions could still have large errors!

Also check RMSE or MAE for actual prediction errors.

â Myth 3: Low RÂ² means the model is useless

Reality: In some fields (like stock prediction), even RÂ² = 0.05 can be valuable if it gives a slight edge.

Context matters more than the absolute value.

â Myth 4: RÂ² tells you if the model is correct

Reality: RÂ² only measures fit, not whether the relationship is real or causal.

You could have high RÂ² with a completely wrong model!

ð Adjusted RÂ² - The Better Metric

Adjusted RÂ² = 1 - [(1 - RÂ²) Ã (n - 1) / (n - p - 1)]
where n = sample size, p = number of predictors

ð¯ Why Adjusted RÂ² is Better

  • Penalizes unnecessary features: Adding useless variables decreases Adjusted RÂ²
  • Fair comparison: Can compare models with different numbers of features
  • Prevents overfitting: Encourages simpler models
  • More realistic: Better estimate of true model performance
# Comparing RÂ² vs Adjusted RÂ² import numpy as np from sklearn.linear_model import LinearRegression from sklearn.metrics import r2_score np.random.seed(42) n_samples = 100 # True relationship: y = 2*x1 + 3*x2 + noise X_useful = np.random.randn(n_samples, 2) y = 2 * X_useful[:, 0] + 3 * X_useful[:, 1] + np.random.randn(n_samples) * 0.5 # Model 1: Only useful features model1 = LinearRegression() model1.fit(X_useful, y) r2_1 = r2_score(y, model1.predict(X_useful)) # Calculate Adjusted RÂ² n = len(y) p = X_useful.shape[1] adj_r2_1 = 1 - (1 - r2_1) * (n - 1) / (n - p - 1) print("Model 1: Only Useful Features") print(f"Features: 2") print(f"RÂ²: {r2_1:.4f}") print(f"Adjusted RÂ²: {adj_r2_1:.4f}") # Model 2: Add 5 random (useless) features X_random = np.random.randn(n_samples, 5) X_combined = np.hstack([X_useful, X_random]) model2 = LinearRegression() model2.fit(X_combined, y) r2_2 = r2_score(y, model2.predict(X_combined)) p2 = X_combined.shape[1] adj_r2_2 = 1 - (1 - r2_2) * (n - 1) / (n - p2 - 1) print("\nModel 2: Useful + 5 Random Features") print(f"Features: 7") print(f"RÂ²: {r2_2:.4f} (increased!)") print(f"Adjusted RÂ²: {adj_r2_2:.4f} (decreased!)") print("\nð¯ Key Insight:") print("RÂ² went UP (seems better) but Adjusted RÂ² went DOWN (actually worse)!") print("The random features added no real value, just overfitting.")
